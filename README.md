Device configuration for yoshino platform
=========================================

This is the Havoc OS device configuration for the Yoshino platform.

Devices based on Yoshino platform:

* Xperia XZ Premium and Dual (maple)
* XPeria XZ1 (poplar)
* Xperia XZ1 Compact (lilac)
